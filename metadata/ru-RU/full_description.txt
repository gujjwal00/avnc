AVNC - это VNC-клиент с открытым исходным кодом для Android. Он позволяет удаленно управлять любым устройством, на котором работает VNC-сервер.

Особенности:

- Material Design (с темной темой)
- Настраиваемые жесты
- Тщательное кодирование
- Виртуальные клавиши
- Режим "картинка в картинке"
- Режим только для просмотра
- Обнаружение сервера Zeroconf
- Поддержка TLS (AnonTLS, VeNCrypt)
- SSH туннель (VNC через SSH)
- Импорт/экспорт серверов
- Поддержка ретранслятора VNC
- Синхронизация буфера обмена с сервером
